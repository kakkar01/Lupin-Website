# LUPIN — 2045

> Premium ultra-minimal futuristic landing page. Brutalist + cyberpunk editorial design.

**Stack:** Next.js 15 · TypeScript · Tailwind CSS · Framer Motion · Vercel

---

## Local Development

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Production Build

```bash
npm run build
npm start
```

## Vercel Deployment

1. Push to GitHub
2. Import repo at [vercel.com/new](https://vercel.com/new)
3. Framework: **Next.js** (auto-detected)
4. Click **Deploy** — no env vars needed

---

## Design

- Dark mode only — black base, monochrome palette
- Blurred grayscale background orb
- Animated film grain (canvas)
- Cursor glow tracking effect
- Framer Motion fade-ins and floating motion
- Corner bracket decorations
- `Space Grotesk` typography
